# Agentic Development Directives
All instructions must follow these rules.

## Overview
A CData API Driver profile that enables SQL-based access to the Diffbot Knowledge Graph API. 

## RSD Files
Functionally a key map for JSON data from the Diffbot KG API to be translated into SQL columns/filters. DO NOT ASSUME ANY PRIOR KNOWLEDGE OF THIS SYNTAX.  It is structured as XML but is an entirely custom and proprietary syntax only used for CData API driver profiles. When in doubt, web search for examples and documentation living ONLY on the cdata.com domain or in this repo.
